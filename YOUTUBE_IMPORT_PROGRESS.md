Прогресс хранится в самой БД (поле `Exercise.youtube_id`): `NULL` — ещё не обработано,
`""` — обработано, видео не найдено, непустая строка — найден `youtube_id`.
Этот файл — просто человекочитаемая заметка на случай новой сессии без памяти разговора.

## Как продолжить импорт

```
flyctl ssh console -a energydess-tools -C "python import_youtube_videos.py"
```

Каждый запуск обрабатывает до 90 упражнений (дневная квота YouTube Data API v3 — 10000
unit/сутки, поиск стоит 100 unit). **Важно**: квота общая на ключ за календарный день —
не запускать скрипт локально и на проде в один день, лимит легко превысить.

Требует подтверждения пользователя перед каждым запуском — это запись в боевую БД на
Fly.io, auto-mode classifier блокирует прямой production-exec без явного "да".
